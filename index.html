Level Up with JavaScript

Hello and welcome to my JavaScript tutorial "Level up with JavaScript".
In this tutorial you will be introduced to some of the basic JavaScript programming concepts.
The content outine in this tutorial directly follows that of FreeCodeCamp.org.
To get started, here is an example of a basic JavaScript program called "Hello World".

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

Now that you have seen what JavaScript is and an example of how to write a simple program, lets level up!

Comments, Declaring Variables, Storing, Assigning, and Initializing Values

Comments

Comments are the notes about your code and describes what it does
JavaScript will ignore everything after "//"" for in-line and /*the is for multiline

// This is an in-line comment.
/* This is a
multi-line comment */

Declaring Variables

Variables allow computers to store and use data. JavaScript uses these eight data types: undefined, null, boolean, string, symbol, bigint, number,and object.
These variables must be declared by the "var" keyword and we always end our JavaScript statments with semicolons ";".
Variables can have numbers, letters, $, or _ . They cannot contain spaces or start with numbers.

var level;

Storing Values

Values can be stored as a varible with the assignment operator (=). In this example I declare the variable "level" and set in to the value "1".
Now each time "level" appears in the code, the program will treat it as "1".

var level;
level = 1;

Assigning Values

Next the assignment operator can be used to assign the same value from the previous variable to another variable. With the below example, now JavaScript_level and level are both the same value of 1.

var level;
level = 1;
var JavaScript_level;
JavaScript_level = level;

Initializing Values

Initialize allows us to assign an initial value to a variable as it is declared.

var level = 1;

Uninitialized Variables, Case Sensitive Variables, Adding Two Numbers, Subtracting Numbers, Multiplying Two Numbers

Uninitialized Variables

When a variable is not set to a value, it is defaulted as "undefined". Not a number(NaN) will be your output if you do not assign a value and try to do a math operation.

var level; alert ( level === undefined )

Case Sensitive Variables

In JavaScript case senitivity matters. "LEVEL" is not the same as "level". Best practice is to use all lower case in single word variables.
In multi-word variables use lower case for the first word, and capitalize the first letter of any subsequent words. This is called camelCase.

var levelUp; var levelUpYourSkills;

Adding Two Numbers

Adding numbers is just as easy as using a (+) between numbers. the result will show as // assigned (number)

level = 1+1; // assigned 2

Subtracting Numbers

The same applies with subtraction, but indtead you use (-).

hitPointsTotal = 10 - 1; // assigned 9

Multiplying Two Numbers

When multiplying nubers we use the (*) operator

damage = 6 * 6; // assigned 36

Dividing Numbers, Increments, Decrements, Multiplying Decimals, and Dividing Decimals

Dividing Numbers

The operator for dividing numbers is the (/) symbol.

experiencePerCharacter = 100 / 5; // assigned 20

Increments

Increments are used to simply add one to a variable. To do this we use the (++) operator.

greatAxe++; (is the same as) greatAxe = greatAxe + 1;

Decrements

Like increments, decrements are used to instead subtract one from a variable . This is fone using the (--) operator.

strength--; (is the same as) strength = strength - 1;

Multiplying Decimals

In JavaScript decimal numbers can also be stored in variables. they are often referred to as floating point numbers aka floats. Keep in mind that these numbers will not be exact and will possibly lead to rounding or accuracy errors.
Decimals can also be multiplied with the (*) operator as well.

Var experiencePerEnemy = 10.5 * 10; // assigned 105

Dividing Decimals

The same applies to dividing decimals.

var experiencePerCharacter = 10.8 / 2; // assigned 5.4

Finding Remainder, Compound Assignment Augmented Addition, C.A.A Subtraction, C.A.A Multiplication, and C.A.A Division

Finding Remainder

When dividing numbers that do not divide as a whole number, the remainder operator (%) can be used to find how much remains after the the division.

var remainder = 13 % 2; var remainder = 1;

Compound Assignment Augmented Addition

Compounded assignment uses the (+=) operator to do the mathmatical operation and then assignment in one step. The mathmatical operation to the right of the equal sign is always performed first and follows the order of operations.

var spellDamage = 10 spellDamage += 5 console.log(spellDamge); // this will return 15

Compound Assignment Augmented Subtraction

Compounded assignments can also be used with augmented subtraction using the (-=) operator.

var hitPoints = 20; hitPoints -= 5 consol.log(hitPoints); // this wil return 15

Compound Assignment Augmented Multiplication

By using the (*=) operator we can use compunded assignment for augmented multiplication.

var criticalHit = 2 criticalHit *= 10 console.log(criticalHit); // this will return 20

Compound Assignment Augmented Division

By using the (/=) operator we can use compounded assignment for augmented division.

var moveSpeed = 30 moveSpeed /= 2 console.log(moveSpeed); // the will return 15

Declare String Variables, Escaping Literal Quotes in Strings, Quoting Strings with Single Quotes, Escape Sequences in Strings, and Concatenating Strings with Plus Operator

Declare String Variables

A string variable is when zero or more characters are enclosed in quotes.

var character = "Tempest Cleric"

Escaping Literal Quotes in Strings

Escaping literal quotes are used when you need to use quotes inside of the quotes of the declared string. This is done using a backslash before both of the actual quotations.

var magicAxe = "Greataxe \"of the Brute\""; (When printed this would look like; Greataxe "of the Brute")

Quoting Strings with Single Quotes

When quoting, single quotes or double quotes may be used. If this is done however, care needs to be taken to avoid ending the string with a none paired single quotation character. Again the backslash can be used before the quotation character to escape it and prevent the error.

errorString = 'The barkeep looked over and asked, "who's responsible for this barbarian?".' correctString = 'The paladin replied, "must be that rouge\'s over there".'

Escape Sequences in Strings

Other than to escape quotes, there are also some special characters that we need to use the backslash to escape. When we do this make sure not to add any spaces between escape sequences or words.
Below is a list of some of these possible characters;

  • \' single quote
  • \" double quote
  • \\ backslash
  • \n newline
  • \r carriage return
  • \t tab
  • \b word boundary
  • \f form feed
var character = "Druid\n\tLevel\One" // This would appear as
Druid
Level One

Concatenating Strings with Plus Operator

Concanating strings uses the (+) concanating operator to add strings together. This is used to build a new string. There will be no spaces unless specifically added.

var character = "Moon " + "Druid"; // The new string would look like this; Moon Druid

Concatenating Strings with Plus Equals Operator, Constructing Strings with Variables, Appending Variables to Strings, Find the Length of a String, Use Bracket Notation to Find the First Character in a String

Concatenating Strings with Plus Equals Operator

As we did with the compounded assignment (+=) operator before, now we will use it to concatenate a string on an existing variable. Again spaces only exist if we add them.

var iroh = "dragon, "; dragon += "of the west."; // iroh would now be "dragon, of the west"

Constructing Strings with Variables

In JavaScript it is common to build longer, more complex strings. To do this we will be using the concatenation operator (+) to insert one or more variables to construct the string.

var mySpell = "magic missile"; spellDescription = "I cast the spell " + mySpell + ", and three glowing darts home in on my target."; // spellDescription will now be "I cast the spell magic missile, and three glowing darts home in on my target."

Appending Variables to Strings

Variables can also be appended to strings using the (+=) operator.

var alignmentTwo = "good" var alignment = "chaotic" alignment += alignmentTwo; // aligment is now "chaotic good"

Find the Length of a String

To find the length of a string we use (.length) after the string but before the end (;). Length is given in the number of characters. It can be used on string variables or string literals.

Example one; "Tabletop game".length; // length would be 13 Example two; var game = "Dungeons and Dragons" var gameLength = game.length; // now gameLength is equal to 20

Use Bracket Notation to Find the First Character in a String

In JavaScript counting starts at 0, and is referred to as Zero-based indexing. By using bracket notation ([]) we can get any character at a specific index in a string.

var character = "Wizard"; var firstLetter = character[0]; // the firstLetter is "W", also class[1] would have been "i" etc.

String Immuntability, Use Bracket Notation to Find the Nth Character in a String, Use Bracket Notation to Find the Last Character in a String, Use Bracket Notation to Find the Nth-to-Last Character in a String

Understand String Immuntability

Immutable means a string cannot be altered once assigned. They way we make changes to strings is to assign with a new string.

var name = "Lizard"; name[0] = "W"; // The is is an example of the wrong way var name = "Lizard"; name = "Wizard"; // This a correct example that reassigns "Lizard" and makes it "Wizard"

Use Bracket Notation to Find the Nth Character in a String

If we want to find which character of a string is sitting at a particular value, we again use bracket notation with the number position we want to find [Nth].

var character = "Druid"; var fourthLetterOfDruid = character[3]; // fourthLetterOfDruid is "i"

Use Bracket Notation to Find the Last Character in a String

When we want to find the last character of a string, we can subtract one from the string's length.

var spell = "Fireball" spell[spell.length - 1]; // the last letter will be "l"

Use Bracket Notation to Find the Nth-to-Last Character in a String

Just as we did before to find the Nth and the last character in a string, we can combine the two to find the last to Nth character.

var enemy = "Goblin Mage"; var secondToLastLetter = enemy[enemy.length - 2]; // the second to last letter is "g"

Store Multiple Values in One Variable Using JavaScript Arrays, Nest One Array Within Another Array, Access Array Data With Indexes, Modify Array Data

Store Multiple Values in One Variable Using JavaScript Arrays

Multiple values can be stored together by enclosing them with brakets []. these can be any combinatiion of strings and/or numbers.

var array = ["Natural", 20]

Nest One Array Within Another Array

Nesting arrays within other ray is also possible by using brackets inside of brackets. T his is referred to as multi-dimensional arrays.

var multiDimensionalArray = [["Hit points", 30], ["Damage Taken", 15]];

Access Array Data With Indexes

Indexes are how we access data inside arrays. They use the same bracket notion as strings but instead are specifiing an entry in an array. Arrays use Zero-based indexing (starts the count from zero) just like strings.

var diceArray = [2,5,6,4,1,1]; diceArray[3]; // this index would be 4

Modify Array Data

As we learned before, strings can not be changed. The entries of arrays are able to be changed however, and this is referred to as mutable.

var diceArray = [2,10,8,7]; diceArray[0] = 4; // now the 2 in our array will be 4

Access Multi-Dimensional Arrays With Indexes, Manipulate Arrays With Push(), Manipulate Arrays with Pop(), Manipulate Arrays With Shift(), Manipulate Arrays With Unshift()

Access Multi-Dimensional Arrays With Indexes

Multi-dimensional arrays can be refered to as an array of arrays. Each set of brackets is a level where the outermost set of brackets are the first level.

var diceArray = [ [18,5,1], [6,10,20], [2,7,15], [[4,6,3], 19, 9] ]; diceArray[0]; // this is [15,5,1] diceArray[3]; // this is [[4,6,3], 19, 9] diceArray[3][0][2]; // this would equal 3

Manipulate Arrays With Push()

Push (.push()) is used to move one or more data parameters to the end of an array.

var diceRoll = ["Roll D6", 4,5,2,6] diceRoll.push(1,5) // diceRoll would now be ["Roll D6", 4,5,2,6,1,5]

Manipulate Arrays with Pop()

There is another way data can be changed in an array and this is with the .pop() function. Using .pop() removes the last element from an array and returns it.

var moonBeam = [8,5,10]; var firstEnemy = moonBeam.pop() cosole.log(firstEnemy); // this will return 10 console.log(moonBeam); // this will now return [8,5]

Manipulate Arrays With shift()

If we want to manipulate the first element instead of the last, we instead use the .shift() function.

var partyComposition = [["Druid", 8 ], ["Fighter", 10], ["Paladin, 10"]] var splitParty = partyComposition.shift() console.log(splitParty); // splitParty will now equal ["Druid", 8 ] console.log(partyComposition); // partyComposition will now equal ["Fighter", 10], ["Paladin, 10"]

Manipulate Arrays With Unshift()

As we learned before with .push() to add an element to the end of an array, .unshift() can be used to add an element to the beginning of an array.

var inventory = ["cloak", "magic ring", "long sword"] inventory.shift(); // inventory now equals ["magic ring", "long sword"] inventory.unshift("health potion"); // now inventory equals ["health potion", "magic ring", "long sword"]
